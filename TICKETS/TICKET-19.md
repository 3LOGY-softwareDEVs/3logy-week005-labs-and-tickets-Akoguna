## 🎯 OBJECTIVE

---

## 📋 REQUIREMENTS

- [ ] Fetch weather data from a free API (e.g., Open-Meteo or wttr.in)

- [ ] useEffect with proper dependency array
- [ ] Auto-update every 60 seconds (with cleanup)

**Suggested free API (no key needed):**
```
https://wttr.in/{city}?format=j1
```
